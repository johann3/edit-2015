The EdIT summer school team built a Lego Mindstorms EV3 robot and a corresponding webshop. The robot, named D0-G3bot, receives a signal from our webshop, fetches the desired product and brings it back.

System core
  • Execution on Java EE platform
  • The system core includes a connection with our Lego robot and other external systems
  • The system's functionality is available via REST services and WebSocket
Web application
  • Developed using AngularJS
  • Includes support for websites with a responsive design
  • Usage of REST andWeb Socket for displaying information and interaction with the system core
Robot
  • Lego Mindstorms EV3
  • WiFi interface (connects the robot with the system core)
Fields: Internet of Things (IoT)
Technologies: Java, AngularJS, HTML5, web services
